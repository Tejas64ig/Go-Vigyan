# Go Vigyan - Sacred Farm OS

A futuristic, full-stack farm management application built with Glassmorphism, React (Vite), and Node.js.

## Modules Included:
- **Client**: React 18 frontend with Tailwind CSS and Framer Motion context.
- **Server**: Node.js + Express backend with MongoDB schemas.

## Running Locally:
```bash
npm install
npm run dev
```
